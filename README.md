## 自用的页面

我个人一直使用的自用页面，觉得很方便，有浏览器的地方就可以用，现在分享给小伙伴，没有什么技术含量，完全静态，整体大小不过百KB，你也可以下载到电脑里自用。

## 首页是搜索页

首页是极简的多引擎搜索页面，输入内容后，鼠标放到搜索按钮悬浮选择使用哪个搜索，你可设为主页使用，右上角的菜单按钮可以跳转到其它页面。

## 网站导航
